3. 기술 스택

레이어기술
FrontendNext.js (App Router), TypeScript, Supabase Auth, Vercel 배포
BackendSpring Boot, Spring Modulith(이벤트 아웃박스), 헥사고날 아키텍처, Gradle 멀티모듈
데이터PostgreSQL(Flyway), Redis(재고 예약 Lua Script, ShedLock 분산락)
메시징Kafka(KRaft 로컬 / Strimzi Operator on k8s) — 주문 이벤트 외부화
결제Toss Payments(실 PG 연동)
배포/인프라Docker Compose(로컬), Kustomize + kind(CI) / Oracle Cloud OKE(운영), SOPS+age 시크릿 암호화
관측성OpenTelemetry SDK/Collector, Tempo(트레이스), Loki(로그), Prometheus(메트릭, native OTLP), Grafana
CI/CDGitHub Actions — 빌드/테스트, GHCR 멀티아크 이미지 푸시, kind 스모크 배포
